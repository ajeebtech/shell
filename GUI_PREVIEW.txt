╔════════════════════════════════════════════════════════════════════════╗
║                     DEMONSTRATION FLOW                                 ║
╚════════════════════════════════════════════════════════════════════════╝

STEP 1: Launch GUI
   → python3 sandbox_gui.py
   → Window opens with orange border
   → Header shows "SANDBOXED SHELL ENVIRONMENT"

STEP 2: Show Banner
   → Shell loads and shows security banner
   → All security features listed with ✓ marks
   → Sandbox directory path displayed

STEP 3: Run Safe Commands (GREEN)
   → help              (shows all commands)
   → ls                (lists sandbox directory)
   → echo "hi" > f.txt (creates file)
   → cat f.txt         (reads file)

STEP 4: Try Blocked Commands (RED)
   → sudo ls           [SANDBOX BLOCKED]
   → rm -rf /          [SANDBOX BLOCKED]
   → cd /etc           [SANDBOX BLOCKED]

STEP 5: Show Statistics
   → stats             (displays execution stats)
   → Shows runtime, executed count, blocked count

STEP 6: Exit
   → exit              (shows final stats)
   → Window closes gracefully
