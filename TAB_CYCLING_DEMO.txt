HOW IT WORKS
─────────────────────────────────────────────────────────────

1. Type a few letters (or nothing)
2. Press TAB to see the first match
3. Press TAB again to see the next match
4. Keep pressing TAB to cycle through ALL matches
5. It loops back to the first match after showing all

RESETTING THE CYCLE
─────────────────────────────────────────────────────────────

The completion resets when you:
• Type a new character
• Delete a character (backspace)
• Press Enter

Example:
  Type: c[TAB]        → cat
  Press: [TAB]        → cd
  Type: l             → cdl (now typing again)
  Press: backspace    → cd
  Press: [TAB]        → cat (resets, starts from beginning)
